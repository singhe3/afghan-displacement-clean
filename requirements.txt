# Afghanistan Displacement Analysis

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Analysis of conflict-induced displacement in Afghanistan.

## 📋 Overview

This project analyzes the relationship between conflict (fatalities) and displacement in Afghanistan using causal inference methods.

## 🎯 Research Questions

1. **DAG 1**: How does location and date affect displacement?
2. **DAG 2**: How do fatalities affect displacement?

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/afghan-displacement-clean.git

# Run the analysis
